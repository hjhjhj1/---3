请在项目下的 login 目录下新建一个 components 文件夹，并添加该组件。该组件用于在获取验证码后显示倒计时，并支持重新发送，需满足以下要求：
1. 基础行为
初始状态为“发送验证码”按钮，点击后触发发送逻辑，并进入倒计时。
倒计时期间按钮置灰，显示剩余秒数（如“59秒后重试”）。
倒计时结束后恢复为“重新发送”按钮，可再次点击。
2. 组件 Props
表格
复制
Prop 名	类型	默认值	说明
onResend	() => void	-	必填，点击发送/重新发送时触发。
countdown	number	60	倒计时秒数。
text	{ default: string; active: string; }	{ default: '发送验证码', active: '秒后重试' }	按钮文案。
3. 特殊 Case
3.1 用户刷新页面
倒计时进行中途若刷新页面，应从本地缓存（如 sessionStorage）恢复剩余时间，继续倒计时。
3.2 多次快速点击
在倒计时进行期间，禁止重复触发 onResend，需防抖处理。
4. 样式与适配
按钮宽度固定，居中显示，倒计时期间文字颜色变灰。
移动端按钮高度不小于 44px，字体不小于 16px，避免点击区域过小。
5. 使用场景
组件完成后，请在 /login 页面下的验证码输入区域下方使用该组件。
